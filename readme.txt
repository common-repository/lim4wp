Provides a button for upload LIM books in a page or post.


== Description ==


Will give your website visitors the ability to play with your educational *[LIM](http://goo.gl/KXziJ)* materials. By default the files will be uploaded to the subfolder called 'lim' in the upload folder set in your preferences.


== Installation ==


1. Upload **Lim4wp** to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress


**Usage:**

To upload your LIM book in a page or post use the red "lim" button of the editor. You will have option to specify parameters like embedded object dimensions and fullscreen.


**Requirements:**

LIM books need to be uploaded in compressed zip file format. To uncompress uploaded data Unix server with "unzip" package installed is required or at least PHP 5.2 version which has the zip extension activated by default. Flash Player 9.0.24 or greater.
